# Jspreadsheet test
to test the functionalties of jspreadsheets you may run the project with the following steps

1. go to ./jspreadsheet folder and run "npm install" and then "npm run build:watch"
2. install the extention in you vscode https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
2. open the project in vscode
3. right click in index.html and select "open in live server", this should open your default browser and show some tables

now you can use ./index.js to use the local ./jspreadsheet in you machine. any changes you make in ./jspreadsheet/index.js will be reflected in index.js. You can even debug in the ./jspreadsheet/index.js
